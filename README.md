# Insurance Claims Analytics Platform

## Overview
This project demonstrates an end-to-end Azure-style data engineering pipeline for insurance claims analytics.

## Tech Stack
- Azure Data Factory
- Azure Databricks
- PySpark
- SQL
- Delta Lake
- Power BI
- GitHub

## Planned Architecture
Raw Data -> Bronze -> Silver -> Gold -> Dashboard

## Repository Structure

- data/raw -> source CSV files
- notebooks -> PySpark ingestion and transformation scripts
- sql -> analytics queries
- docs -> project overview and architecture notes
